# Сайт-визитка: консультации по YouTube-продвижению

Минимальный учебный проект без сборки и зависимостей.

## Что внутри

- `index.html` — структура страницы и контент.
- `styles.css` — оформление, карточки и адаптация под телефон.

## Как открыть сайт

1. Откройте папку проекта.
2. Дважды кликните по файлу `index.html`.
3. Сайт откроется в браузере.

## Что можно настроить под себя

- Ссылка на Telegram в финальном блоке (`https://t.me/your_username`).
- Имя в блоке «Обо мне» (сейчас: Сергей).
- При необходимости — текст и контакты под ваши услуги.
