# Мой первый проект для Codex

Это учебный репозиторий, в котором я учусь пользоваться OpenAI Codex.

## Цель

Научиться работать с Codex: создавать файлы, читать код, вносить маленькие изменения и понимать, что происходит в проекте.

## Первый проект

Я хочу создать простой сайт-визитку для консультаций по YouTube-продвижению.

Сайт должен быть простым:
- без React;
- без npm;
- без сложной сборки;
- только HTML и CSS;
- чтобы его можно было открыть через файл index.html в браузере.
